== Description ==


<h1>Ni WooCommerce Product Variations Table Description</h1>
<p>
Change WooCommerce default dropdown to variations table with columns header sorting.
</p>
<p>
Ni WooCommerce Product Variations Table provide the option to customize the columns of variation table like variation image, variation name, product SKU, product price, stock status and stock quantity.
</p>
<h2>Ni WooCommerce Product Variations Table Feature</h2>
<ul>
<li>Change variation dropdown to variation table</li>
<li>Columns header sorting</li>
<li>Ajax add to cart</li>
<li>Setting option for variation table columns customization</li>
<li>Report Summary total sales, total order, recent order, order status report </li>
</ul>
